This option extracts either the Chameleon, Clover or Ozmosis bootlog from the IORegistry and saves it to a file named XXXXX_BootLog.txt, where XXXXX is the name, and possibly revision, of the bootloader. This file is saved in the latest dump/BootLog_Firmware folder.

The logs are dumped using the bdmesg tool by Kabyl, since modified by JrCs from an idea by STLVNUB.
