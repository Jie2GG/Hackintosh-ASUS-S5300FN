This runs a shell script to get an overview of the structure of the local disks and partitions using the system tool diskutil.

The boot sector of each disk and each partition are read and an attempt is made to identify what is there. All non-blank boot sectors are written to a file named with the disk number, Manufactuer/model and size and saved in the latest dump/Disk/Boot Sectors folder.

Bootloader Stage 2 and UEFI boot files are searched for and an attempt is made to identify them, before being listed in Bootloaders.txt and saved in the latest dump/Boot Loaders folder.

Note This scan requires root privileges.
