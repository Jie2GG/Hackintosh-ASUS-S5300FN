Two processes are run with this option:
1 - x86info is executed
2 - AppleIntelInfo.kext is loaded and the result read.

x86info

This binary identifies the current CPU.

Source ProjectOSX

AppleIntelInfo.kext
WARNING!
There is a chance your machine could reboot using this.

Source Piker-Alpha's AppleIntelInfo GitHub repo
Note Root privileges are required to load AppleIntelInfo.kext.
