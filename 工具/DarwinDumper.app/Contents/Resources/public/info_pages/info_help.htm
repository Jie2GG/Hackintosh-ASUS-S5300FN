DarwinDumper is essentially a collection of shell scripts, and executable tools which work together to gather various data from the running system. You select what to dump by ticking the respective checkbox for a category.

Sidebar options:

Symlink Add a symbolic link: /usr/local/bin/darwindumper

Doing this will enable you to initiate DarwinDumper from the command line by typing darwindumper.

A tick will appear beside the menu option title if a valid symlink exists.
Custom Path Set a custom path for saving the dumps.

A tick will appear beside the menu option title if a custom path has been set.
DeSelect All Dumps De-selects all dump checkboxes.
Select All Dumps Selects all dump checkboxes.
Run Begins executing the selected dumps.
Note If you have selected any dumps that require root privileges then you will be asked for your password.
