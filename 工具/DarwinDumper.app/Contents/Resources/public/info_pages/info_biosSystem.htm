The option installs DirectHW.kext and then runs the flashrom utility to read the flash chip and save it to file.

biosbackup.rom (the ROM file) and flashrom_log.txt (the log) are saved in the latest dump/BIOS/System folder.

The flashrom utility can do a lot more and is used for identifying, writing and erasing flash chips also. You can find more information at flashrom.org.

Source Projectosx Forum Topic
Note Root privileges are required to load the DirectHW.kext and run the flashrom tool.
