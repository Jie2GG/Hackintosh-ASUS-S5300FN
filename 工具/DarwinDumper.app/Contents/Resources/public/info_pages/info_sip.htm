System Integrity Protection is a security technology in OS X El Capitan that's designed to help prevent potentially malicious software from modifying protected files and folders on your Mac.

Detailed information can be found online, but for a light intro see Apple's page

Here we use Piker-Alpha's csrstat utility for reading the current state of SIP as used internally by the kernel

Source https://github.com/Piker-Alpha/csrstat

The dump is saved to a file named SIP_status.txt in the latest dump/SIP folder.
