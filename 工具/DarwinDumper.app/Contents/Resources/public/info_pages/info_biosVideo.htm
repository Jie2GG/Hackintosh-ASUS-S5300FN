The option runs the following routines to find and save vbios ROM's.

First, legacy ROM's are searched for in shadow ROM addr 0xC0000 thanks to Andy Vandijck's updated version of dong's RadeonPCI.kext and RadeonDump tool. ATI, NVidia and Intel chipsets can be dumped. Second, any legacy ATY,bin_image ROM's found in IORegistry are saved. Third, a VFCT ACPI table is searched for and scanned for a UEFI GOP BIOS.

The ROM files are saved to dump/BIOS/Video folder with the filename notation beginning with VendorID.DeviceID. If this vendor matches 1002 (ATI/AMD) then Andy Vandijck's radeon_bios_decode tool is run to decode the BIOS dump file.

Lastly, the atomdis binary is run against any ATI ROM files in the hope of disassembling them. And the AnalyseVBIOS tools is run against any Nvidia ROM's to find the address of any VESA tables.
