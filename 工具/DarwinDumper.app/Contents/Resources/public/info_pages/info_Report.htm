The HTML report gives you an alternative method for viewing the dumps.

Enabling this option instructs another shell script to run once all the dumps have completed. It scans the latest dump folder in DarwinDumperReports and constructs an HTML report file, including as much of the dumped information as possible.

You now have the option of choose between a new style report as well as the classic style report, either open of collapsed.

The difference with the new style report is it now utilises the standard jQuery and jQueryUI libraries with the accordion and tabs widgets currently being used. Adopting this approach will make it easier to add other features in the future.

Note Not all the dumped information is included in the HTML report. If asking for help and supplying a DarwinDumper report it would be best to supply the complete folder containing the latest dump.
